Bước ① ② — Tín hiệu thô & Pre-emphasis

Đọc file .wav và lọc tiền nhấn tăng cường tần số cao

🎙️ Bước ①: Đọc tín hiệu
# librosa.load()
x, sr = librosa.load(wav_path, sr=None)

# Kết quả:
x  → numpy array 1D (float32)
sr → 16000 Hz
  • sr = None Giữ nguyên tần số gốc (16kHz)
  • x.shape (num_samples,)
  • Ví dụ 3s (48000,) = 3 × 16000
🔊 Bước ②: Pre-emphasis Filter
# Bộ lọc tiền nhấn
x = librosa.effects.preemphasis(
    x, zi=[0.0]
)
Công thức:
y[n] = x[n] − α · x[n−1]
α = 0.97 (mặc định librosa)

Mục đích: Tăng cường các thành phần tần số cao trong tín hiệu, cải thiện độ phân giải phổ cho giọng nói.

Slide 3/10

Bước ③ ④ ⑤ — STFT → Log (dB) → Lọc tần số

Biến đổi sóng âm 1D thành ma trận phổ tần số-thời gian 2D

# Bước ③: Short-Time Fourier Transform
spec = np.abs(librosa.stft(
    x,
    n_fft=800,          # Số điểm FFT
    hop_length=160,     # = 10ms × 16000Hz → bước trượt
    win_length=640,     # = 40ms × 16000Hz → cửa sổ
    window='hamming'
))                    # → shape: (401, T)

# Bước ④: Chuyển sang thang dB
spec = librosa.amplitude_to_db(spec, ref=np.max)
                             # → shape: (401, T), dải [-80dB, 0dB]

# Bước ⑤: Giữ 200 bins tần số thấp nhất
spec = spec[:200]        # → shape: (200, T)

# Thêm chiều Channel
spec = np.expand_dims(spec, 0) # → shape: (1, 200, T)

Bước ⑥ — Phân đoạn (Segmentation)

Cắt spectrogram thành các đoạn cố định (1, 200, 300) ≈ 3 giây

✂️ Logic phân đoạn
# segment_size = 300 frames
# Mỗi frame = 10ms → 300 × 10ms = 3s

num_segs = math.ceil(T / 300)

for i in range(num_segs):
  if end > T:  # Đoạn cuối
    # Zero-padding
    np.pad(..., "constant")

# Output: (N, 1, 200, 300)
# → Transpose → (N, 1, 300, 200)
# → Transpose → (N, 1, 200, 300)

Slide 9/10

Dự đoán cấp Câu nói (Utterance-level)

Trung bình hóa xác suất tất cả segments thuộc cùng 1 câu nói

📐 Công thức
ŷu = argmaxc [ 1/|Su| × Σ log P(c|s) ]
Su = tập các segments thuộc câu nói u
P(c|s) = xác suất lớp c cho segment s (log-softmax)
argmaxc = chọn lớp có xác suất trung bình cao nhất

# Câu nói dài 8s → 3 segments
